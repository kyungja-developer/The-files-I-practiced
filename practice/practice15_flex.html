flex-flow 
          flex-flow: 주축 여러줄묶음 ;
          (flex-diretion : items의 주 축(mai-axis)을 설정 / 기본값 : row  - float:left, right같은 것
          ( row : 좌-우 / row-reverse : 우-좌 / column : 상-하 / column-reverse : 하-상 정렬)

          flex-wrap : items의 여러 줄 묶음(줄 바꿈 설정 / 기본값 : nowrap)
          (nowrap : 모든 items를 여러 줄로 묶지 않음(한 줄에 표시) /기본값임
          wrap : items를 여러 주로 묶음
          wrap-reverse : items를 wrap의 역 방향으로 여러 줄로 묶음)

flex-grow

        모든 items의 총 증가 너비 (flex-grow)에서 각 item의 증가 너비의 비율 만큼 너비를 가질 수 있다. 
        전체 container의 너비를 몰라도 item들의 너비를 비율적으로 설정할 수 있다.
        기본값은 0
1
2

flex-shrink

        item이 감소하는 너비의 비율 설정. 숫자 클수록 더 많이 감소. 
        item이 가변 너비가 아니거나, 값이 0일 경우 효과X.
        flex-basis의 영향을 받는다!

flex-basis

        공간배분전 item의 기본 너비를 설정. 
        기본값 auto
