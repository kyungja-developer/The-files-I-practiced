그리드 함수들-minmax

행,열의 '최소/최대'크기 정의

        첫번째 인수는 '최솟값', 두번째 인수는 '최댓값' : minmax(최솟값, 최댓값) <-첫번째 열
        grid-template-rows, grid-template-columns, grid-auto-rows, grid-auto-columns에서 사용
        일반 요소에 min-width, max-width 속성을 동시에 지정하는 것과 유사

        아래 예시)grid-template-columns : minmax(100px, 1fr) minmax(200px, 1fr);
