grid-template-areas(item속성)

        지정된 그리도 영역 이름(grid-area)을 참조해 그리드 템플릿을 생성한다.
        각 영역의 이름을 정한다.
HEADER
MAIN
ASIDE
FOOTER
NAME
AGE
        grid-area는 grid-row-start/end , grid-column-start/end의 단축속성도 됨. 
        하지만!
        grid-template-area로 영역 이름을 설정할 경우
        grid-row / grid-column 개념은 무시됨.

        grid-area: < grid-row-start > / < grid-column-start > / < grid-row-emd > / < grid-column-start >
        'row column row colum 순서' ( row row column column 순서 아님! )
        ex)
        grid-row: 2/3;
        grid-column: span 2 / -1;
        =
        grid-area: 2 / span 2 / 3 / -1;


        또는 원래대로
        grid-area: 영역이름;
