gird-template-rows

        명시적 행(track)의 크기를 정의
        동시에 라인(Line)의 이름도 정의
        fr(fraction, 공간 비율) 단위 사용 가능. 
        repeat() 함수 사용 가능.

        행의 '이름' 설정 가능
        ex) grid-template-rows: [first] 100px [second] 200px [third];
        한 라인에 중복된 이름 지정 가능(띄어쓰기로)
        ex) grid-template-rows: [row1-start] 100px [row-1end row2-start] 200px [row2-end];

        이런것도 가능 정해진 단위 반복
        grid-template-rows: repeat(4, 100px 200px 50px);
        /*100px 200px 50px 100px 200px 50px 100px 200px 50px 100px 200px 50px 
        100px 200px 50px을 순서대로 4번 반복*/
